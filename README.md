# Integracja Systemów Informatycznych - Laboratorium 1

Lokalna zmiana dokumentacji

**Celem niniejszego projektu** jest praktyczne opanowanie systemu kontroli wersji Git, platformy GitHub oraz przygotowanie lokalnego środowiska programistycznego dla frameworka Django.

### Wymagania projektowe:
* Zainstalowany system kontroli wersji **Git**.
* Środowisko programistyczne **Python** w wersji 3.x.
* Skonfigurowany dostęp przez protokół **SSH** do serwisu GitHub.

### Podstawowe komendy:
Aby uruchomić serwer deweloperski, należy użyć komendy `python manage.py runserver` wewnątrz aktywnego środowiska wirtualnego.

### Dokumentacja:
Więcej informacji na temat użytego frameworka można znaleźć w [oficjalnej dokumentacji Django](https://docs.djangoproject.com/en/stable/).
